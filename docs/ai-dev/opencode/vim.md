## Vim 模式

OpenCode 内置 Vim 模式，支持纯键盘操作。

## 启用 Vim 模式

### 配置文件

```yaml
vim:
  enabled: true
```

### 命令行

启动时指定：

```bash
opencode --vim
```

## 常用快捷键

### 模式切换

| 快捷键 | 说明 |
|--------|------|
| `i` | 进入插入模式 |
| `Esc` | 返回 Normal 模式 |
| `:` | 进入命令模式 |

### Normal 模式

| 快捷键 | 说明 |
|--------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w/b` | 前进/后退一个词 |
| `0/$` | 行首/行尾 |
| `gg/G` | 文件开头/结尾 |
| `dd` | 删除整行 |
| `yy` | 复制整行 |
| `p` | 粘贴 |
| `u` | 撤销 |
| `Ctrl+r` | 重做 |

### 插入模式

| 快捷键 | 说明 |
|--------|------|
| `Ctrl+h` | 删除前一个字符 |
| `Ctrl+w` | 删除前一个词 |
| `Ctrl+u` | 删除到行首 |
| `Esc` | 退出插入模式 |

### 命令模式

| 命令 | 说明 |
|------|------|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 不保存强制退出 |
| `:set number` | 显示行号 |
| `/关键词` | 搜索 |

## 退出 Vim

在 Normal 模式下：

- `:q` - 退出
- `:wq` - 保存并退出
- `:q!` - 强制退出不保存

或者按 `ZZ` 保存退出。
