## Introduction

**OpenCode** is a terminal AI programming assistant built with Go, featuring an interactive TUI interface. It supports multiple AI models (OpenAI, Claude, Gemini, etc.), can execute commands, search files, modify code, and integrates with Vim editor mode and LSP for intelligent code completion.

### Use Cases

- Developers
- Programmers
